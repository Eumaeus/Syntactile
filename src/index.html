Stage 2: Define Verbal Units

Define a verbal unit by selecting Syntactic Type, Semantic Type, and Nesting Level (1–5). Then click Confirm.

ID Syntactic Type Semantic Type Level Actions

Stage 3: Assign Tokens to Verbal Units

Select a verbal unit from the menu “Assign to Verbal Unit:”. Click on tokens to add them to that verbal unit. A second click will remove them from the verbal unit. Use the “Assign to Verbal Unit” to change which verbal unit you are assigning tokens to. Repeat until all tokens have been assigned to one or more units.

Stage 4: Syntactic Relationship Analysis

CITE2-URN:

Define syntactic relationships to build a top-down graph rooted at '0: Sentence Root'. For each token, select linked tokens (Node 1, Node 2, including '0: Sentence Root') and enter relations (e.g., 'sentence adverbial' for γὰρ, 'Unit Verb' for the main verb). A main verb linking to '0: Sentence Root' indicates asyndeton; a linking word (e.g., γὰρ) indicates none. Export as CEX to save.
