Stage 1: Define Verbal Units

Define a verbal unit by selecting Syntactic Type, Semantic Type, and Nesting Level (1–5). Then click Confirm. Select the new verbal unit from the menu “Assign to Verbal Unit:”. Assign tokens to this unit; tokens can belong to multiple units for nested structures. Edit or delete units as needed. Define a second unit if appropriate; select it from the menu; and assign tokens to it. Repeat until all tokens have been assigned to one or more units. Token 0 (Sentence Root) is reserved for Stage 2.

Stage 2: Syntactic Relationship Analysis

CITE2-URN:

Define syntactic relationships to build a top-down graph rooted at '0: Sentence Root'. For each token, select linked tokens (Node 1, Node 2, including '0: Sentence Root') and enter relations (e.g., 'sentence adverbial' for γὰρ, 'Unit Verb' for the main verb). A main verb linking to '0: Sentence Root' indicates asyndeton; a linking word (e.g., γὰρ) indicates none. Export as CEX to save.
